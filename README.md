# Emiliano Nakayama Portfolio

Sitio estatico minimalista para presentar el perfil Java Backend de Emiliano Nakayama.

## Abrir localmente

Abrir este archivo en el navegador:

```text
portfolio/index.html
```

No requiere instalacion ni servidor local.

## Personalizar

- Editar enlaces y textos en `index.html`.
- Ajustar estilos en `styles.css`.
- Reemplazar la imagen principal en `assets/backend-workspace.png`.
- Para agregar proyectos, duplicar el bloque `project-layout` en `index.html` y cambiar titulo, descripcion, metricas y enlaces.
- Si agregas una foto personal, usar una imagen real, profesional y sobria. Conviene ubicarla en la seccion de perfil o contacto, no como elemento principal del hero.

## Publicar

Puede subirse como sitio estatico a GitHub Pages, Netlify, Vercel o cualquier hosting que sirva archivos HTML/CSS/JS.
